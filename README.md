# ReportRadar

A responsive news website built for the W06 Project, using HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Features
- Homepage with breaking news, featured articles, and newsletter subscription.
- Category pages for World and Tech news.
- About page with mission and team info.
- Contact page with a form for feedback.
- Search functionality, social media sharing, and comment sections.
- Responsive design, accessible HTML, and optimized images.

## Setup
1. Clone the repository: `git clone https://github.com/<username>/reportradar.git`
2. Open `index.html` in a browser or host via GitHub Pages.
3. Source royalty-free images for `images/` folder and update `references.html`.

## Hosting
- Live at: `https://<username>.github.io/reportradar`
- Enable GitHub Pages in repository settings.
